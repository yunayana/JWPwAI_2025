# Języki wysokiego poziomu w aplikacjach internetowych

Repozytorium zawiera komplet rozwiązań z laboratoriów realizowanych w ramach przedmiotu  
**„Języki wysokiego poziomu w aplikacjach internetowych”**.

Celem projektu jest praktyczne poznanie nowoczesnych technologii webowych oraz ich zastosowania w tworzeniu aplikacji internetowych – zarówno po stronie backendu, jak i frontendu.

## Struktura repozytorium

Każde laboratorium znajduje się w osobnym katalogu i stanowi niezależny projekt

## Wykorzystane technologie

- **Node.js** – środowisko uruchomieniowe JavaScript
- **React** – biblioteka do budowy interfejsów użytkownika
- **Next.js** – framework React z obsługą routingu i SSR
- **JavaScript (ES6+)** – nowoczesne standardy języka
- **HTML5 & CSS3** – struktura i stylowanie aplikacji
